# VIM

Just a repo that stores frequently used commands from Vim.

## ⌨️ Key

#### Enter edit mode

This "key" will let you enter edit mode in Vim.

```
i
```

#### Leave

This "key" will let you exit edit mode in Vim.

```
esc
```

#### Save update

This "key" will save the file and exit Vim editor.

```
:wq
```
